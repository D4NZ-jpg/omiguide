Este sitio está pensado para que puedas escribir en **MDX** (archivos en `content/guide/`) y que el frontend se actualice automáticamente.

- Crea carpetas para **secciones** (ej: `blog/`, `secciones/`).
- Crea archivos `index.mdx` para la portada de una sección.
- Agrega posts/páginas como `mi-post.mdx`.
